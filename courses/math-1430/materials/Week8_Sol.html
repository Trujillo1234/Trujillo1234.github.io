Calculus 2 • Indefinite Integrals, U-Substitution, Definite Integrals, & Net Change

1. Antiderivatives & Basic Indefinite Integrals

Evaluate each indefinite integral.

  1. $$\int \left(4x^3-7x+2\right)\,dx$$

    Integrate term-by-term using the power rule.

    $$\int \left(4x^3-7x+2\right)\,dx = \int 4x^3\,dx - \int 7x\,dx + \int 2\,dx$$ $$= x^4 - \frac{7}{2}x^2 + 2x + C$$
    Answer: $$x^4-\frac{7}{2}x^2+2x+C$$
  2. $$\int \left(6\sqrt{x}-\frac{5}{x}\right)\,dx$$

    Rewrite $\sqrt{x}$ as $x^{1/2}$ and use the logarithm rule for $\dfrac{1}{x}$.

    $$\int \left(6x^{1/2}-5x^{-1}\right)\,dx = 6\int x^{1/2}\,dx - 5\int x^{-1}\,dx$$ $$= 6\left(\frac{x^{3/2}}{3/2}\right) - 5\ln|x| + C = 4x^{3/2} - 5\ln|x| + C$$
    Answer: $$4x^{3/2}-5\ln|x|+C$$
  3. $$\int \left(e^x+3e^{2x}\right)\,dx$$

    Split the integral into two pieces.

    $$\int \left(e^x+3e^{2x}\right)\,dx = \int e^x\,dx + 3\int e^{2x}\,dx$$

    The first part is basic:

    $$\int e^x\,dx=e^x$$

    For the second part, use substitution.

    Let $u=2x$. Then $du=2\,dx$, so $dx=\dfrac{1}{2}\,du$.

    $$3\int e^{2x}\,dx = 3\int e^u\left(\frac{1}{2}\right)du = \frac{3}{2}\int e^u\,du$$ $$= \frac{3}{2}e^u + C = \frac{3}{2}e^{2x}+C$$

    So the full antiderivative is

    $$\int \left(e^x+3e^{2x}\right)\,dx = e^x+\frac{3}{2}e^{2x}+C$$
    Answer: $$e^x+\frac{3}{2}e^{2x}+C$$
  4. $$\int \left(\sin x-2\cos x\right)\,dx$$

    Use the basic trig antiderivatives.

    $$\int \sin x\,dx = -\cos x,\qquad \int (-2\cos x)\,dx = -2\sin x$$ $$\int \left(\sin x-2\cos x\right)\,dx = -\cos x - 2\sin x + C$$
    Answer: $$-\cos x-2\sin x+C$$
  5. $$\int \sec^2(x)\,dx$$

    This is one of the basic trig antiderivatives.

    $$\int \sec^2(x)\,dx = \tan x + C$$
    Answer: $$\tan x + C$$
  6. $$\int \sec(x)\tan(x)\,dx$$

    This is another basic trig antiderivative.

    $$\int \sec(x)\tan(x)\,dx = \sec x + C$$
    Answer: $$\sec x + C$$
  7. $$\int \frac{1}{1+x^2}\,dx$$

    This is the standard arctangent antiderivative.

    $$\int \frac{1}{1+x^2}\,dx = \arctan(x)+C$$
    Answer: $$\arctan(x)+C$$
  8. Find $F(x)$ such that $F'(x)=\dfrac{3}{x-5}$.
    (Write your answer as a single logarithm.)

    We want an antiderivative of $\dfrac{3}{x-5}$.

    Factor out the $3$ first:

    $$\int \frac{3}{x-5}\,dx = 3\int \frac{1}{x-5}\,dx$$

    Now use substitution.

    Let $u=x-5$. Then $du=dx$.

    $$3\int \frac{1}{x-5}\,dx = 3\int \frac{1}{u}\,du = 3\ln|u|+C$$ $$=3\ln|x-5|+C$$
    Answer: $$F(x)=3\ln|x-5|+C$$

2. Spot the U-Substitution

Evaluate each integral using $u$-substitution. Indicate the choice of $u$ clearly.

  1. $$\int (2x+7)^6\,dx$$

    Let $u=2x+7$. Then $du=2\,dx$, so $dx=\dfrac{1}{2}du$.

    $$\int (2x+7)^6\,dx = \int u^6\left(\frac{1}{2}\right)du = \frac{1}{2}\int u^6\,du$$ $$= \frac{1}{2}\cdot \frac{u^7}{7}+C = \frac{(2x+7)^7}{14}+C$$
    Answer: $$\frac{(2x+7)^7}{14}+C$$
  2. $$\int \frac{1}{x\ln(x)}\,dx$$

    Let $u=\ln(x)$. Then $du=\dfrac{1}{x}\,dx$.

    $$\int \frac{1}{x\ln(x)}\,dx = \int \frac{1}{u}\,du = \ln|u|+C$$ $$= \ln|\ln(x)|+C$$
    Answer: $$\ln|\ln(x)|+C$$
  3. $$\int \frac{(\ln x)^7}{x}\,dx$$

    Let $u=\ln x$. Then $du=\dfrac{1}{x}\,dx$.

    $$\int \frac{(\ln x)^7}{x}\,dx = \int u^7\,du = \frac{u^8}{8}+C$$ $$= \frac{(\ln x)^8}{8}+C$$
    Answer: $$\frac{(\ln x)^8}{8}+C$$
  4. $$\int x\sin(x^2)\,dx$$

    Let $u=x^2$. Then $du=2x\,dx$, so $x\,dx=\dfrac{1}{2}du$.

    $$\int x\sin(x^2)\,dx = \frac{1}{2}\int \sin(u)\,du = -\frac{1}{2}\cos(u)+C$$ $$= -\frac{1}{2}\cos(x^2)+C$$
    Answer: $$-\frac{1}{2}\cos(x^2)+C$$
  5. $$\int \cos(5x-1)\,dx$$

    Let $u=5x-1$. Then $du=5\,dx$, so $dx=\dfrac{1}{5}du$.

    $$\int \cos(5x-1)\,dx = \frac{1}{5}\int \cos(u)\,du = \frac{1}{5}\sin(u)+C$$ $$= \frac{1}{5}\sin(5x-1)+C$$
    Answer: $$\frac{1}{5}\sin(5x-1)+C$$
  6. $$\int \frac{2x+1}{x^2+x}\,dx$$

    Let $u=x^2+x$. Then $du=(2x+1)\,dx$.

    $$\int \frac{2x+1}{x^2+x}\,dx = \int \frac{1}{u}\,du = \ln|u|+C$$ $$= \ln|x^2+x|+C$$
    Answer: $$\ln|x^2+x|+C$$
  7. $$\int \tan(x)\,dx$$

    First rewrite $\tan x$ using sine and cosine.

    $$\tan x=\frac{\sin x}{\cos x}$$

    Now use substitution.

    Let $u=\cos x$. Then $du=-\sin x\,dx$, so $\sin x\,dx=-du$.

    $$\int \tan(x)\,dx = \int \frac{\sin x}{\cos x}\,dx = \int \frac{\sin x}{u}\,dx = \int \frac{-1}{u}\,du$$ $$= -\int \frac{1}{u}\,du = -\ln|u|+C$$ $$= -\ln|\cos x|+C$$

    This is equivalent to $\ln|\sec x|+C$.

    Answer: $$-\ln|\cos x|+C$$
  8. $$\int \sqrt{1-4x}\,dx$$

    Let $u=1-4x$. Then $du=-4\,dx$, so $dx=-\dfrac{1}{4}du$.

    $$\int \sqrt{1-4x}\,dx = -\frac{1}{4}\int u^{1/2}\,du = -\frac{1}{4}\cdot \frac{u^{3/2}}{3/2}+C$$ $$= -\frac{1}{6}u^{3/2}+C = -\frac{1}{6}(1-4x)^{3/2}+C$$
    Answer: $$-\frac{1}{6}(1-4x)^{3/2}+C$$
  9. $$\int (3x^2)e^{x^3}\,dx$$

    Let $u=x^3$. Then $du=3x^2\,dx$.

    $$\int (3x^2)e^{x^3}\,dx = \int e^u\,du = e^u + C$$ $$= e^{x^3}+C$$
    Answer: $$e^{x^3}+C$$

3. Definite Integrals & Substitution with New Limits

Evaluate each definite integral. If you change variables, change the limits too.

  1. $$\int_{0}^{2} (4x+1)^5\,dx$$

    Let $u=4x+1$. Then $du=4\,dx$, so $dx=\dfrac{1}{4}du$.

    Change the limits:

    $$x=0 \Rightarrow u=1,\qquad x=2 \Rightarrow u=9$$ $$\int_{0}^{2}(4x+1)^5\,dx = \frac{1}{4}\int_{1}^{9}u^5\,du = \frac{1}{4}\left[\frac{u^6}{6}\right]_{1}^{9}$$ $$= \frac{1}{24}\left(9^6-1\right)$$
    Answer: $$\frac{9^6-1}{24}$$
  2. $$\int_{1}^{e} \frac{1}{x}\,dx$$

    Use the logarithm antiderivative.

    $$\int_{1}^{e}\frac{1}{x}\,dx = \left[\ln|x|\right]_{1}^{e} = \ln(e)-\ln(1)$$ $$= 1-0 = 1$$
    Answer: $$1$$
  3. $$\int_{0}^{\pi/2} \frac{\sin x}{1+\cos x}\,dx$$

    Let $u=1+\cos x$. Then $du=-\sin x\,dx$.

    Change the limits:

    $$x=0 \Rightarrow u=2,\qquad x=\frac{\pi}{2} \Rightarrow u=1$$ $$\int_{0}^{\pi/2}\frac{\sin x}{1+\cos x}\,dx = -\int_{2}^{1}\frac{1}{u}\,du = \int_{1}^{2}\frac{1}{u}\,du$$ $$= \left[\ln|u|\right]_{1}^{2} = \ln 2 - \ln 1 = \ln 2$$
    Answer: $$\ln 2$$
  4. $$\int_{1}^{4} \frac{\ln(x)}{x}\,dx$$

    Let $u=\ln x$. Then $du=\dfrac{1}{x}\,dx$.

    Change the limits:

    $$x=1 \Rightarrow u=0,\qquad x=4 \Rightarrow u=\ln 4$$ $$\int_{1}^{4}\frac{\ln(x)}{x}\,dx = \int_{0}^{\ln 4}u\,du = \left[\frac{u^2}{2}\right]_{0}^{\ln 4}$$ $$= \frac{(\ln 4)^2}{2}$$
    Answer: $$\frac{(\ln 4)^2}{2}$$
  5. $$\int_{0}^{1} x\sqrt{1+x^2}\,dx$$

    Let $u=1+x^2$. Then $du=2x\,dx$, so $x\,dx=\dfrac{1}{2}du$.

    Change the limits:

    $$x=0 \Rightarrow u=1,\qquad x=1 \Rightarrow u=2$$ $$\int_{0}^{1}x\sqrt{1+x^2}\,dx = \frac{1}{2}\int_{1}^{2}u^{1/2}\,du$$ $$= \frac{1}{2}\left[\frac{u^{3/2}}{3/2}\right]_{1}^{2} = \frac{1}{3}\left[u^{3/2}\right]_{1}^{2}$$ $$= \frac{1}{3}\left(2^{3/2}-1\right) = \frac{2\sqrt{2}-1}{3}$$
    Answer: $$\frac{2\sqrt{2}-1}{3}$$

4. Net Change Theorem (Rates $\rightarrow$ Total Change)

Use the idea total change = integral of the rate. For total distance, integrate the absolute value of velocity.

  1. A particle moves on a line with velocity $v(t)=4t-6$ (m/s).

    1. Net displacement on $[0,3]$.

      $$\text{Net displacement} = \int_{0}^{3}(4t-6)\,dt$$ $$= \left[2t^2-6t\right]_{0}^{3} = (18-18)-0 = 0$$
      Answer: $$0\text{ meters}$$

    2. Total distance traveled on $[0,3]$.

      First find where the velocity changes sign:

      $$4t-6=0 \Rightarrow t=\frac{3}{2}$$

      So we split the interval at $t=\dfrac{3}{2}$.

      $$\text{Total distance} = \int_{0}^{3}|4t-6|\,dt = -\int_{0}^{3/2}(4t-6)\,dt + \int_{3/2}^{3}(4t-6)\,dt$$ $$= -\left[2t^2-6t\right]_{0}^{3/2} + \left[2t^2-6t\right]_{3/2}^{3}$$ $$= -\left(\frac{9}{2}-9\right) + \left(0-\left(\frac{9}{2}-9\right)\right)$$ $$= \frac{9}{2}+\frac{9}{2}=9$$
      Answer: $$9\text{ meters}$$
  2. A tank is being filled at a rate of $r(t)=2+\sqrt{t}$ (gal/min). If the tank starts with $15$ gallons, how much is in the tank after $9$ minutes?

    The amount added over the first $9$ minutes is

    $$\int_{0}^{9}(2+\sqrt{t})\,dt = \int_{0}^{9}\left(2+t^{1/2}\right)\,dt$$ $$= \left[2t+\frac{2}{3}t^{3/2}\right]_{0}^{9} = 18+\frac{2}{3}(27)$$ $$=18+18=36\text{ gallons}$$

    Since the tank started with $15$ gallons,

    $$\text{Amount after 9 minutes}=15+36=51$$
    Answer: $$51\text{ gallons}$$
  3. A car’s velocity (mph) is given by the graph below.
    Use geometry on the graph. There are three triangular regions: two above the $x$-axis and one below the $x$-axis.

    1. Approximate the net displacement on $[0,7]$.

      We first find the equations of the line segments so we can locate the $x$-intercepts exactly.

      First segment: from $(0,2)$ to $(2,-1)$

      $$m=\frac{-1-2}{2-0}=-\frac{3}{2}$$ $$y=mx+b \quad\Rightarrow\quad y=-\frac{3}{2}x+b$$

      Use the point $(0,2)$:

      $$2=-\frac{3}{2}(0)+b \Rightarrow b=2$$

      So the equation is

      $$y=-\frac{3}{2}x+2$$

      Now set $y=0$ to find the intercept:

      $$0=-\frac{3}{2}x+2$$ $$\frac{3}{2}x=2$$ $$x=\frac{4}{3}$$

      Second segment: from $(2,-1)$ to $(5,3)$

      $$m=\frac{3-(-1)}{5-2}=\frac{4}{3}$$ $$y=mx+b \quad\Rightarrow\quad y=\frac{4}{3}x+b$$

      Use the point $(2,-1)$:

      $$-1=\frac{4}{3}(2)+b$$ $$-1=\frac{8}{3}+b$$ $$b=-\frac{11}{3}$$

      So the equation is

      $$y=\frac{4}{3}x-\frac{11}{3}$$

      Now set $y=0$ to find the intercept:

      $$0=\frac{4}{3}x-\frac{11}{3}$$ $$\frac{4}{3}x=\frac{11}{3}$$ $$x=\frac{11}{4}$$

      Third segment: from $(5,3)$ to $(7,0)$

      $$m=\frac{0-3}{7-5}=-\frac{3}{2}$$ $$y=mx+b \quad\Rightarrow\quad y=-\frac{3}{2}x+b$$

      Use the point $(5,3)$:

      $$3=-\frac{3}{2}(5)+b$$ $$3=-\frac{15}{2}+b$$ $$b=\frac{21}{2}$$

      So the equation is

      $$y=-\frac{3}{2}x+\frac{21}{2}$$

      Its $x$-intercept is found by setting $y=0$:

      $$0=-\frac{3}{2}x+\frac{21}{2}$$ $$\frac{3}{2}x=\frac{21}{2}$$ $$x=7$$

      So the three triangular regions are:

      • $A_1$: above the axis from $x=0$ to $x=\frac{4}{3}$
      • $A_2$: below the axis from $x=\frac{4}{3}$ to $x=\frac{11}{4}$
      • $A_3$: above the axis from $x=\frac{11}{4}$ to $x=7$

      Now find each triangle’s area.

      Triangle $A_1$:

      base $=\dfrac{4}{3}$ and height $=2$

      $$A_1=\frac{1}{2}\left(\frac{4}{3}\right)(2)=\frac{4}{3}$$

      Triangle $A_2$:

      base $=\dfrac{11}{4}-\dfrac{4}{3}=\dfrac{33}{12}-\dfrac{16}{12}=\dfrac{17}{12}$ and height $=1$

      $$A_2=\frac{1}{2}\left(\frac{17}{12}\right)(1)=\frac{17}{24}$$

      Triangle $A_3$:

      base $=7-\dfrac{11}{4}=\dfrac{28}{4}-\dfrac{11}{4}=\dfrac{17}{4}$ and height $=3$

      $$A_3=\frac{1}{2}\left(\frac{17}{4}\right)(3)=\frac{51}{8}$$

      For net displacement, areas above the axis are positive and areas below the axis are negative:

      $$\text{Net displacement}=A_1-A_2+A_3$$ $$=\frac{4}{3}-\frac{17}{24}+\frac{51}{8}$$ $$=\frac{32}{24}-\frac{17}{24}+\frac{153}{24}$$ $$=\frac{168}{24}=7$$
      Answer: $$7\text{ miles}$$

    2. Approximate the total distance traveled on $[0,7]$.

      For total distance, we add all three triangle areas, even the one below the axis.

      $$\text{Total distance}=A_1+A_2+A_3$$ $$=\frac{4}{3}+\frac{17}{24}+\frac{51}{8}$$ $$=\frac{32}{24}+\frac{17}{24}+\frac{153}{24}$$ $$=\frac{202}{24}=\frac{101}{12}$$ $$\approx 8.42$$
      Answer: $$\frac{101}{12}\text{ miles} \approx 8.42\text{ miles}$$
  4. A population grows at a rate of $P'(t)=120e^{0.05t}$ (people/year). If $P(0)=5000$, find $P(10)$.

    By net change,

    $$P(10)=P(0)+\int_{0}^{10}P'(t)\,dt = 5000+\int_{0}^{10}120e^{0.05t}\,dt$$

    To integrate $e^{0.05t}$, use substitution.

    Let $u=0.05t$. Then $du=0.05\,dt$, so $dt=\dfrac{1}{0.05}du=20\,du$.

    Change the limits:

    $$t=0 \Rightarrow u=0,\qquad t=10 \Rightarrow u=0.5$$ $$5000+\int_{0}^{10}120e^{0.05t}\,dt = 5000+\int_{0}^{0.5}120e^u(20)\,du$$ $$=5000+2400\int_{0}^{0.5}e^u\,du =5000+2400\left[e^u\right]_{0}^{0.5}$$ $$=5000+2400(e^{0.5}-1)$$
    Answer: $$P(10)=5000+2400\left(e^{0.5}-1\right)\approx 6556.10\text{ people}$$
