Taylor Polynomials

Calculus 2 Week 1 Notes Instructor Copy
Learning Objectives
  • Construct a Taylor Polynomial of degree $n$ for a function $f(x)$ centered at $x=a$.
  • Derive the coefficient formula involving factorials using repeated differentiation.
  • Estimate the error of an approximation using the Lagrange Error Bound.

1. Motivation: From Lines to Curves

The Goal: How do calculators compute values for complicated functions like $e^x$, $\sin(x)$, or $\ln(x)$? They approximate them using simple polynomials (using only $+$, $-$, $\times$).

Recall (Linear Approximation): The best approximation of a function $f(x)$ near $x=a$ using a line is the tangent line.

Degree 1 Approximation (Tangent Line) $$ P_1(x) = f(a) + f'(a)(x-a) $$ Matches height and slope at $x=a$.

2. Deriving the Quadratic Approximation

Lines are straight, but functions curve. To capture the concavity (bending), we need a polynomial of degree 2 (a parabola).

Let's guess the form centered at $x=a$:

$$ P_2(x) = A(x-a)^2 + B(x-a) + C $$

We need $P_2$ to match $f$ in value, slope, and concavity at $x=a$.

Step 1: Differentiate $P_2(x)$

  • $P_2(x) = A(x-a)^2 + B(x-a) + C$
  • $P_2'(x) = 2A(x-a) + B$
  • $P_2''(x) = 2A$

Step 2: Solve for Coefficients at $x=a$

  • $P_2(a) = C \quad \implies \quad C = f(a)$
  • $P_2'(a) = B \quad \implies \quad B = f'(a)$
  • $P_2''(a) = 2A \quad \implies \quad 2A = f''(a) \implies {A = \frac{f''(a)}{2}}$
Degree 2 Approximation $$ P_2(x) = f(a) + f'(a)(x-a) + \frac{f''(a)}{2}(x-a)^2 $$

3. The General Taylor Polynomial

Extending this pattern to degree $n$, we require the $k$-th derivative of the polynomial to match the $k$-th derivative of the function. The repeated use of the Power Rule generates a factorial in the denominator.

Definition: Taylor Polynomial of Degree n $$ P_n(x) = f(a) + f'(a)(x-a) + \frac{f''(a)}{2!}(x-a)^2 + \dots + \frac{f^{(n)}(a)}{n!}(x-a)^n $$
$$ P_n(x) = \sum_{k=0}^{n} \frac{f^{(k)}(a)}{k!} (x-a)^k $$

4. Guided Example

Problem: Find the 3rd degree Taylor Polynomial ($P_3(x)$) for $f(x) = \ln(x+1)$ centered at $a=0$.

k Derivative $f^{(k)}(x)$ Value at $x=0$ Coefficient $\frac{f^{(k)}(0)}{k!}$
0 $\ln(x+1)$ $\ln(1) = 0$ $0$
1 $(x+1)^{-1}$ $1$ $1/1! = 1$
2 $-(x+1)^{-2}$ $-1$ $-1/2! = -1/2$
3 $2(x+1)^{-3}$ $2$ $2/3! = 1/3$

Solution: Construct the polynomial using the coefficients calculated above.

$$ P_3(x) = 0 + 1(x) - \frac{1}{2}x^2 + \frac{1}{3}x^3 $$ $$ \boxed{ P_3(x) = x - \frac{1}{2}x^2 + \frac{1}{3}x^3 } $$

5. Error Analysis (Lagrange Error Bound)

Since $P_n(x)$ is only an approximation, there is an error $R_n(x) = f(x) - P_n(x)$.

Theorem If $|f^{(n+1)}(t)| \le M$ for all $t$ between $a$ and $x$, then: $$ |R_n(x)| \le \frac{M}{(n+1)!} |x-a|^{n+1} $$

Intuition: The error depends on the "next" derivative (how fast the rate of change is changing) and the distance from the center.
