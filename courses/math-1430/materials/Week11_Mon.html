Integration by Parts

Calculus 2 • Notes
Learning Objectives
  • Use integration by parts to find antiderivatives.
  • Apply the tabular method for polynomials to perform repeated integration by parts efficiently.

1. The Formula and Steps

We use integration by parts to find antiderivatives. The standard formula is:

$$ \int u \, dv = uv - \int v \, du $$
  1. Pick $u$ and identify $dv$.
  2. Find $du$ using differentiation.
  3. Find $v$ using integration.
  4. Plug into the formula. → Repeat if needed.

2. Introductory Examples

Ex: $\int 2x e^x \, dx$
$u = 2x$
$du = 2 \, dx$
$dv = e^x \, dx$
$v = e^x$
$$ \int 2x e^x \, dx = 2x e^x - \int 2e^x \, dx $$
$$ = 2x e^x - 2e^x + C $$

check work:

$$ \require{cancel} (2x e^x - 2e^x + C)' = \cancel{2e^x} + 2x e^x - \cancel{2e^x} + 0 $$ $$ = 2x e^x \quad \checkmark $$
Ex: $\int x^2 e^x \, dx$
$u = x^2$
$du = 2x \, dx$
$dv = e^x \, dx$
$v = e^x$
$$ \int x^2 e^x \, dx = x^2 e^x - \int 2x e^x \, dx $$ $$ = x^2 e^x - (2x e^x - 2e^x) + c $$
$$ = x^2 e^x - 2x e^x - 2e^x + c $$

3. Repeated Integration by Parts

Ex: $\int (x^2 + 3x)\cos(x) \, dx$
$u = x^2 + 3x$
$du = 2x + 3 \, dx$
$dv = \cos(x) \, dx$
$v = \sin(x)$
$$ = (x^2 + 3x)\sin(x) - \int (2x + 3)\sin(x) \, dx $$
$u = 2x + 3$
$du = 2 \, dx$
$dv = \sin(x) \, dx$
$v = -\cos(x)$
$$ = (x^2 + 3x)\sin(x) - \left( (2x + 3)(-\cos(x)) - \int -2\cos(x) \, dx \right) $$
$$ = (x^2 + 3x)\sin(x) + (2x + 3)\cos(x) - 2\sin(x) + c $$

4. The Tabular Method

The Tabular Method ($u = \text{polynomial}$)

$u$ $dv$
$+$ $x^2 + 3x$ $\cos(x)$
$-$ $2x + 3$ $\searrow$ $\sin(x)$
$+$ $2$ $\searrow$ $-\cos(x)$
$-$ $0$ $\searrow$ $-\sin(x)$

$\int (x^2 + 3x)\cos(x) \, dx =$

$$ (x^2 + 3x)\cos(x) + (2x + 3)\cos(x) - 2\sin(x) + c $$

5. Tabular Method Example & Antiderivative Note

Ex: $\int (x^2 + 3x + 1)e^{2x} \, dx =$
$u$ $dv$
$+$ $x^2 + 3x + 1$ $e^{2x}$
$-$ $2x + 3$ $\searrow$ $e^{2x}/2$
$+$ $2$ $\searrow$ $e^{2x}/4$
$-$ $0$ $\searrow$ $e^{2x}/8$
$$ \Rightarrow (x^2 + 3x + 1)\frac{e^{2x}}{2} - (2x + 3)\frac{e^{2x}}{4} + 2\frac{e^{2x}}{8} + c $$
Note: Update List of antideriv.
$$ \int e^{kx} \, dx = \frac{e^{kx}}{k} + c $$

6. Integration with Logarithms

Ex: $\int (x^3 + 5x^2 + x + 1)\ln(x) \, dx$
$u = \ln(x)$
$du = \frac{1}{x}$
$dv = x^3 + 5x^2 + x + 1 \, dx$
$v = \frac{x^4}{4} + \frac{5x^3}{3} + \frac{x^2}{2} + x$
$$ = \ln(x)\left(\frac{x^4}{4} + \frac{5x^3}{3} + \frac{x^2}{2} + x\right) - \int \frac{x^3}{4} + \frac{5x^2}{3} + \frac{x}{2} + 1 \, dx $$
$$ = \ln(x)\left(\frac{x^4}{4} + \frac{5x^3}{3} + \frac{x^2}{2} + x\right) - \left(\frac{x^4}{16} + \frac{5x^3}{9} + \frac{x^2}{4} + x\right) + c $$
