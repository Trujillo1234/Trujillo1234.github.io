Module 1

Logic of Atomic Sentences

Arguments in the blocks language, validity, consequence, and the way atomic statements support formal reasoning.

Guiding Questions

  • What do we mean by logical consequence?
  • What is a valid argument?
  • What is a sound argument?
  • Are valid arguments always sound?
  • Are sound arguments always valid?
  • What is =Intro, and what is =Elim?
  • How can we prove that a sentence is not a consequence of others?
  • What sentences are logical truths?
  • What arguments involving quantification are valid?

Objectives

  • Apply the system F to construct formal proofs in the Fitch format.
  • Identify the premises and conclusion of an argument.
  • Classify an argument as valid or invalid.
  • Classify an argument as sound or unsound.
  • Construct formal proofs in Fitch using introduction and elimination rules for quantifiers.
  • Construct formal proofs in Fitch using the rule of general conditional proof .
  • Apply the axiomatic method : Zermelo-Fraenkel Set Theory ( ZF )
  • Peano Arithmetic ( PA )
  • Shape Axioms
  • Apply axiomatic set theory.
