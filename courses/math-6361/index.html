Course Map

MATH 6361: Mathematical Logic.

A public-facing reconstruction of the Blackboard module sequence. The course moves from atomic sentences and propositional logic into formal proof, soundness, completeness, quantification, axiomatic set theory, and incompleteness.

Course Design

Each module was built around guiding questions, learning outcomes, reading notes, video notes, homework, discussion, and quiz practice. That structure makes the course inspectable: visitors can see what students are asked to do and why.

What Comes Next

The Blackboard export preserved the HTML module materials, but the video links need a second pass because Blackboard packages them as course resources rather than ordinary web links. I have kept this page ready for those links as they are recovered.

Module Sequence

Module 0: Atomic Sentences

Constants, predicates, function symbols, terms, and translation between English and first-order atomic sentences.

Module 1: Propositional Logic

Arguments, validity, soundness, counterexamples, and early formal proof in Fitch style.

Module 2: Boolean Connectives

Truth-functional connectives, complex sentences, truth tables, and translation.

Module 3: The Logic of Boolean Connectives

Tautologies, logical truths, logical consequence, equivalence, and truth-table methods.

Module 4: Formal Proof Methods for Boolean Logic

Introduction and elimination rules, proof by contradiction, proof by cases, and subproofs.

Module 5: The Logic of Conditionals

Material conditionals, biconditionals, English translation, and proof rules for arrows.

Module 6: Soundness and Completeness

The relationship between proof systems and semantic consequence in propositional logic.

Module 7: Midterm Review

Review and synthesis of propositional and predicate-logic foundations before the midterm.

Module 9: The Logic of Quantification

Well-formed formulas, free and bound variables, quantified sentences, and truth in first-order structures.

Module 10: Proof Methods for Quantifiers

Universal and existential reasoning, prenex form, informal quantified proof, and the axiomatic method.

Module 11: Formal Proof Methods for Quantifiers

Fitch proofs using quantifier introduction and elimination rules, including identity.

Module 12: Axiomatic Set Theory and Mathematical Structures

Extension, separation, power sets, unions, intersections, relations, functions, and mathematical structures.

Modules 13/14: Completeness and Incompleteness

Soundness, completeness, incompleteness, and the boundary between formal proof and mathematical truth.
